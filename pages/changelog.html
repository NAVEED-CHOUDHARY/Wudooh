Changelog


Version 2.0 (Not finished)


The long awaited major update that adds highly requested features, a sleek modern user interface, this web page you're reading, even more amazing fonts, new browsers support and a few other things as well!

The user interface now has a modern design based on Google's Material Design

The sliders for font size and line height now go up to 300% instead of the previous 200%

Wudooh now has its own web page for help, updates and download links

Wudooh is now available for Edge(Chromium), Opera and as a Chromium extension to be used in Chromium browsers such as Brave, Vivaldi and Epic.

You can now set site specific settings in addition to the global settings!

You can now export and import your settings so you can safely use them across browsers.

Wudooh now has even more awesome fonts to use in addition to the already huge library of fonts!

We've made tons of cleanups and optimizations behind the scenes to make sure the extension is as fast and useful as possible while being completely unnoticeable



Translations are still on the way

Custom font upload (a feature requested by some) is being actively explored but may take a long time (but it's not fully off the list yet (Thanks Amen Jlili:P))
