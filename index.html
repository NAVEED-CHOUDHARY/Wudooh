Frequently Asked Questions


What languages does Wudooh support?


Wudooh, supports any language that uses Arabic script, this means languages like Arabic, Persian, Urdu, Pashto, Kurdish, Uyghur, Sindhi and basically any language that uses a variety of Arabic script.
This is because Wudooh modifies text containing characters in the Arabic Unicode range .



What fonts does Wudooh have?


Wudooh, has over 35 free and open source fonts, including the no-font font (lol wut).



How can I request a feature or font?


Wudooh is constantly being improved, and your contribution is always welcome! You can request features or fonts on the GitHub issues page or by sending us an email at whitecrescent.uk@gmail.com.



How can I help in the development of Wudooh?


Your contribution is very much welcomed and encouraged! Currently Wudooh is in need of translators and front end developers, pull requests to fix issues on the GitHub issues page are greatly welcome!



I've found a bug, where can I report it?


You can report bugs either on the GitHub issues page or by sending us an email at whitecrescent.uk@gmail.com. Remember to describe the bug in detail so that it will be easy for us to replicate.



Where can I give feedback or a review?


All feedback is greatly appreciated! The best place to give feedback is by giving Wudooh a review on its store page on the browser you use.
